def elma.api.manager.Manager.emit (   self,

)

Emit an event associated with a name.

Typically, a process would emit events in its update() method using something like the following code"

emit(Event("name", value));

where value is any jsonable value. For example, you can write

emit(Event("velocity", 3.41));
Parameters
eventThe Event to be emitted
